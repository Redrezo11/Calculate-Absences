Overview

The attendance is based on specifications set by the IT/Engineering dept of the Institute of Languages.

To be able to calculate student absences, you will need to run the code snippet generated by the website for the specific class that you are trying to analyze.

In order for this code-snippet to work correctly, you will need to use the developer's console in the Google Chrome web browser and run the code snippet over the page on the Tabuk BlackBloard page that contains the student attendance report data for your class.

It maybe a little complicated to setup, but once that is finished it will be easy to run the code snippet that is saved in your Chrome browser to get the information you want easily, saving time.

This is because the code snippet or program that is run, will produce all the serial numbers of the late and absent students which are easier to look up than student ID's. This will let you update the attendance form on MYUT easily and quickly in most cases. It makes a real difference if you have something like a class of 50 students and maybe only 5 students decide to go absent.

However, if the student list changes for whatever reason, such as a student dropping the class a new student list should be run through the code-generator to update the proper serial numbers for each student.

Initial Setup

  1. Open Google Chrome web browser, if you do not have it please install it here.
  2. Export the student list on MyUT in Microsoft Excel and save it in an easily accessible folder. Make sure to name the file correctly so that it won't cause confusion.
  3. Go back to this website page, open that same file for the class you want to generate a script for and then, press the parse file button, and the proceed to copy the generated code to the windows clipboard by pressing the now visible copy to clipboard button.
  4. Open the javascript developers console. By pressing CTRL-SHIFT-J on the keyboard. If it is not working for you, here is a video, explaining how to reach the location.
  5. Create a new code snippet, give it a good name, and then paste the code into the snippet window.

How to use

  1. You will need to open up the class attendance report on BlackBoard for the specified date that you wish to find student absences from, you will need to open up the "Printable Copy" version of the attendance report instead of the Excel file export. Here is a link if you do not know how to get to this page.
  2. Open up the developers console in Google Chrome by pressing CTRL+SHIFT+J and then navigate to the code-snippets pane and select the specified script for the class. When you have found it, click the play button to run the script.
